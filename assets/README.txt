Westside-style static site skeleton for Groots
--------------------------------------------
Files you can replace:
- assets/hero.jpg
- assets/banner-1.jpg
- assets/product-1.jpg ... product-12.jpg
- css/style.css to edit styles (colors, fonts)
- js/main.js for additional interactions

How to use:
- Unzip the package and open index.html in your browser to preview locally.
- Replace images in /assets with real photos (use same filenames for convenience).
- Update product copy inside HTML files or implement a simple JSON + JS loader if you want dynamic data.
- If you want contact form to work, configure a backend endpoint or use services like Formspree / Netlify Forms.
